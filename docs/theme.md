# Chatsian Theme

## Summary
Chatsian is a calm, focused theme that balances cool teal primaries with warm amber accents. It is optimized for long study sessions and clear hierarchy, with soft backgrounds and crisp foreground contrast.

## Palette
Light mode tokens (OKLCH):
- Background: 0.985 0.01 250
- Foreground: 0.18 0.02 260
- Primary: 0.58 0.16 195
- Accent: 0.92 0.05 70
- Border: 0.9 0.015 250

Dark mode tokens (OKLCH):
- Background: 0.16 0.01 255
- Foreground: 0.98 0.005 250
- Primary: 0.7 0.15 195
- Accent: 0.3 0.06 70
- Border: 1 0 0 / 12%

## Usage
- Theme tokens live in `app/globals.css` under `:root` and `.dark`.
- Sidebar tokens are aligned with the main palette.
- Hero accents use subtle radial gradients with teal highlights.

## Brand Notes
- Primary teal signals progress and action.
- Amber accent highlights focus and insight moments.
- Neutral backgrounds avoid fatigue and keep content legible.
